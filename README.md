# GotADice
A simple project that generates random values.

## Environment setup
Ensure that `Python 3.10+` is installed in your computer.
```bash
python3 -m venv ~/.gotadice
source ~/.gotadice/bin/activate
```

## Run backend app
```bash
flask run --reload
```
